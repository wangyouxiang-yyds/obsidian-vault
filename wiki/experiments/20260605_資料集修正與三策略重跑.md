## 修正三：重建 JSON flat 目錄

**問題**：`JSON_ms6/` 刪除後，三個 yaml 的 `json_dir` 失效；`fuzzy_find_file()` 只做 flat `glob`，無法在 `MS6_sen2like/` 遞迴搜尋。

**解法**：建立新的 flat 目錄，每個 JSON 改用**場景資料夾名稱**命名（避免重名問題）。

```bash
# 兩個場景同日期不同 tile，原 JSON 檔名相同：
# NOAA_Gulf_of_Mexico_High_Confidence_20200611_S2_16RBT/  → S2.json (同名)
# NOAA_Gulf_of_Mexico_High_Confidence_20200611_S2_16RCT/  → S2.json (同名)
# 改為用 scene dir name 命名 → 兩個不同檔名，不覆蓋
```

**輸出**：`/home/alanyh/oil_dataset/new/full_band/MS6_sen2like_JSON/`（439 個 JSON）

**yaml 更新**（3 個全改）：
```yaml
json_dir: '/home/alanyh/oil_dataset/new/full_band/MS6_sen2like_JSON'
```
