# DeepRX + EM v3 全 439 場景無監督油汙偵測實驗

## 實驗背景

延續 [[20260613_iForest_LocalContrast_B4B8B11_全場景實驗總結]] 的結果，該實驗 Micro Recall 僅 0.022。本次採用 **DeepRX + EM** 架構（陳繼藩指導，[[柯弈仲2008_光學衛星影像海洋異常物偵測]] + [[Song_DeepRX_Hyperspectral_Anomaly]] 兩篇論文的融合），結合 VAE encoder 的 latent 表示與 Mahalanobis 距離（RX），並用 EM 自動門檻取代固定 percentile。

實驗目標：在「寧可錯抓不要漏抓」的策略下，把 Recall 提升到能當「人工標記候選提示器」的可用門檻。

方法理論基礎：[[unsupervised_oil_detection]]

---

## Pipeline 流程

```
VRT 場景（10980×10980×8 bands）載入
    ↓
Pass 1: 收集乾淨水體像素（NDWI > 0.05，排雲，排亮 B02 > 0.4）
    ↓
Pass 2: Per-scene VAE 訓練（50 epochs，lambda_1=0.9 重建主導，lambda_2=0.1 KLD）
    ↓
Pass 2.5: 計算背景 latent 統計（μ, Σ⁻¹）
    ↓
方案 C: Coast Buffer（±1 patch 內含陸地的 patch 全跳過）
    ↓
Pass 3: DeepRX 評分（latent space 的 Mahalanobis 距離）
    ↓
方案 A: Pass 3.5 Local Contrast 鄰域 z-score 重排序（±5 patches）
    ↓
Pass 4: EM 自動門檻（2-component GMM 找 Bayes 最小誤差交叉點）
    ↓
驗證：cand_set ∩ oil_set → TP/FP/FN/Recall/Precision/F1
```

### 關鍵設計（v3）

#### 三層遮罩（與 iForest 一致）

1. **Patch 級別 `patch_is_scoreable`**：陸地 > 2%、B02 std > 0.04、NoData > 5% → 跳過整個 patch
2. **Coast Buffer（新方案 C，v3 用 ±1）**：任何 patch 在 ±1 patch 範圍內有陸地 patch → 跳過
3. **像素級別 `loose_mask`**：NoData (B02<0.005) + 雲 (B11>0.05 AND B02>0.20) + 陸地 (B02>0.4) 排除

#### Local Contrast（新方案 A）

不用絕對 Mahalanobis 距離排名，改成「跟 ±5 patch 鄰域比的 z-score」。解決訓練資料污染導致的「整片高分區誤判」問題（沿岸、雲邊整片 RX 分數高 → z-score 拉平 → 排除）。

#### EM 自動門檻

取代固定 top 15%。對分數分布擬合 2-component Gaussian Mixture（背景 + 異常），找兩條 PDF 的 Bayes 最小誤差交叉點當門檻。

## 觀察與結論

### 成功之處

- **DeepRX 解決 iForest 的根本限制**：iForest 用 PCA + 隔離樹學「水體分布」，遇到非高斯背景就失準。DeepRX 用 VAE encoder + KL divergence 強制 latent 服從高斯，剛好滿足 RX 的假設。
- **Local Contrast 是關鍵**：抵消「訓練資料含油汙污染」的副作用——整片偏離 vs 局部突起被分開了。
- **EM 自動門檻配合「錯抓不漏抓」策略**：EM 找的雙峰交叉點通常比固定 top 15% 更激進（更多候選），完全符合本實驗目標。

### 仍存在的問題

- **Precision 仍極低（0.0041）**：1,698 TP 換 414,162 FP，比例 1:244。當「候選提示器」可用，但離自動偵測還很遠。
- **希臘薩龍灣（Agia Zoni）** 仍很弱（AvgR=0.024）：四個場景中有兩個被 Coast Buffer 過度抑制；油汙分布形狀可能特別不利於 z-score 偵測。
- **NOAA Pacific Ocean** 全零（3 場景）：值得單獨檢查影像，可能跟海況、油汙特徵都不一樣。
- **模里西斯** 全零（1 場景）：樣本太少難下結論。

### 與 iForest 的根本差異

- **iForest**：固定 percentile + 全 8 波段 PCA。對「典型異常」靈敏，但對「跟水體光譜接近」的油汙完全沒辦法。
- **DeepRX + EM**：自適應門檻 + VAE encoder 連續 latent 表示。能捕捉到 iForest 漏掉的細微光譜偏離。

## 後續改進方向

- [ ] **提升 Precision**：1:244 的 FP/TP 比例還可以更好。考慮加入 score 二次過濾（例如 patch 內 oil-spectral-index 額外驗證）
- [ ] **救希臘薩龍灣 / Pacific Ocean**：實際看影像確認失敗原因，可能需要場景特定調參或不同特徵
- [ ] **減少 FP 數量**：414k FP 對人工複查還是太多，目標降到 ~50k
- [ ] **比較其他閾值方法**：EM vs Otsu vs 固定 percentile 三者在不同場景的優劣
- [ ] **將 v3 套用到 iForest pipeline**：把 Local Contrast 改成「跟 ±5 patch 比 z-score」也試試
